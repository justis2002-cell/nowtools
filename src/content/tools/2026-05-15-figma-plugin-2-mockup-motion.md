# 피그마 플러그인 추천 2편 - 목업·애니메이션·꾸미기

## 디자인 완성도는 디테일에서 나옵니다
1편에서는 이미지와 아이콘처럼 디자인의 뼈대를 채우는 플러그인을 정리했습니다. 이번 2편에서는 디자인에 생동감과 완성도를 더해주는 플러그인을 모았습니다.

목업으로 결과물을 더 그럴듯하게 보여주고, 모션과 GIF로 움직임을 더하고, 그림자와 그라데이션으로 입체감을 살리는 작업은 사실 별것 아닌 것 같아도 결과물의 인상을 크게 바꿉니다. 상세페이지나 카드뉴스의 완성도를 한 단계 올리고 싶을 때 도움이 되는 플러그인들입니다.

## 1. Mockup Plugin - 디자인을 실제 환경에 입혀보기
디자인한 화면을 노트북, 스마트폰, 태블릿 같은 실제 디바이스 화면에 입혀서 보여주는 플러그인입니다. 같은 디자인이라도 빈 화면 캡처보다 실제 기기에 들어간 모습으로 보여주면 훨씬 그럴듯해 보입니다.

**어떤 작업에 유용한가**
- 포트폴리오에 작업물 보여줄 때
- 클라이언트에게 시안 전달할 때
- 상세페이지 상단의 메인 이미지로 활용할 때
- 웹·앱 디자인 결과물을 SNS에 공유할 때

빈 디자인을 그대로 올리는 것보다 실제 디바이스 안에 들어간 모습으로 보여주면 신뢰도가 올라갑니다.

## 2. Motion - 피그마 안에서 애니메이션 만들기
피그마에서 만든 디자인에 움직임을 더해주는 플러그인입니다. 페이드인, 슬라이드, 회전, 스케일 같은 기본 애니메이션부터 좀 더 복잡한 모션까지 피그마 안에서 직접 설정할 수 있습니다.

**어떤 작업에 유용한가**
- 프로토타입에 움직임을 더해 더 실감 나게 보여줄 때
- 카드뉴스에 들어갈 짧은 모션 만들 때
- 인터랙션 흐름을 클라이언트에게 설명할 때

정적인 디자인만으로 부족할 때, 간단한 움직임 하나만 추가해도 결과물이 살아납니다.

## 3. LottieFiles - 가벼운 벡터 애니메이션
Lottie는 JSON 기반의 가벼운 벡터 애니메이션 파일 형식입니다. LottieFiles 플러그인을 쓰면 수많은 Lottie 애니메이션을 피그마 안에서 검색하고 바로 삽입할 수 있습니다.

**어떤 작업에 유용한가**
- 로딩 아이콘, 체크 마크 같은 작은 모션 추가
- 일러스트에 자연스러운 움직임 더하기
- 웹사이트나 앱에 그대로 적용 가능한 애니메이션 확보

GIF보다 용량이 훨씬 작으면서도 화질 손상이 없어서, 실제 개발 단계까지 그대로 이어 쓸 수 있다는 게 장점입니다.

## 4. GIF 내보내기 플러그인 3종
피그마에서 만든 디자인이나 프레임을 GIF로 내보낼 수 있는 플러그인들입니다. 비슷한 기능이지만 결과물 품질이나 사용법이 조금씩 달라서, 상황에 맞춰 골라 쓰면 됩니다.

- **GIF Export**: 가장 기본적인 GIF 내보내기 플러그인. 단순한 작업에 무난.
- **Export to GIF, Video**: GIF뿐 아니라 영상 형식으로도 내보낼 수 있어 활용 범위가 넓음.
- **GiffyCanvas**: 캔버스 위에서 프레임을 직접 구성해 GIF로 만드는 방식. 좀 더 자유로운 작업이 가능.

**어떤 작업에 유용한가**
- 카드뉴스를 움직이는 GIF로 만들어 SNS에 올릴 때
- 디자인 시안에 짧은 움직임을 더해 메신저로 공유할 때
- 인스타그램 피드용 짧은 모션 콘텐츠 제작

같은 기능처럼 보여도 플러그인마다 결과물 품질이 다르기 때문에, 한두 개 깔아두고 비교해서 쓰면 좋습니다.

## 5. 텍스트 곡선 플러그인 2종 (ARC, Text ARC)
텍스트를 곡선 형태로 휘어주는 플러그인입니다. 피그마 기본 기능만으로는 텍스트를 자연스러운 원형이나 곡선 모양으로 배치하기 어려운데, 이 플러그인들을 사용하면 한 번에 해결됩니다.

- **ARC - Bend your type!**: 텍스트를 자유롭게 휘어주는 기능. 각도 조절 직관적.
- **Text ARC**: 비슷한 기능이지만 인터페이스나 세부 옵션이 조금 다름.

**어떤 작업에 유용한가**
- 원형 로고나 뱃지 디자인
- 카드뉴스 타이틀에 강조 효과를 줄 때
- 일러스트 주변을 감싸는 텍스트 배치
- 도장이나 스탬프 느낌의 디자인

직선으로만 텍스트가 배치되면 단조로워 보일 때, 곡선 텍스트 하나만 추가해도 디자인이 풍부해집니다.

## 6. Beautiful Shadows - 자연스러운 그림자 만들기
피그마 기본 그림자 기능보다 훨씬 자연스럽고 세밀한 그림자를 만들어주는 플러그인입니다. 빛의 위치와 강도를 직관적으로 조절할 수 있어서, 입체감이 살아있는 그림자를 빠르게 만들 수 있습니다.

**어떤 작업에 유용한가**
- 카드 형태 UI에 깊이감 더할 때
- 버튼이나 모달에 자연스러운 그림자 적용
- 상세페이지 제품 이미지에 입체감 줄 때
- 일러스트 요소에 부드러운 그림자 추가

기본 그림자 효과는 평면적으로 보이는 경우가 많은데, 이 플러그인을 사용하면 결과물이 훨씬 고급스러워 보입니다.

## 7. 그라데이션 플러그인 2종 (Accidently Great Gradients, uiGradients)
잘 어울리는 그라데이션 조합을 골라주는 플러그인입니다. 색상 조합을 직접 만들기 어려울 때, 이미 검증된 예쁜 그라데이션을 바로 적용할 수 있습니다.

- **Accidently Great Gradients**: 우연히 발견한 듯한 자연스러운 그라데이션 컬렉션. 트렌디한 조합 많음.
- **uiGradients**: UI 디자인에 적합한 그라데이션 모음. 깔끔하고 실용적인 조합 위주.

**어떤 작업에 유용한가**
- 메인 이미지나 배너 배경 색감 잡을 때
- 카드뉴스 표지 배경
- 버튼·박스 등 UI 요소의 포인트 색상
- 상세페이지 섹션 구분용 배경

색감을 직접 만드는 것보다 빠르고, 결과물도 안정적입니다.

## 마무리
2편에서 소개한 플러그인들은 디자인의 뼈대를 만드는 도구라기보다, **이미 만든 디자인을 한 단계 더 다듬어주는 도구**에 가깝습니다. 목업으로 결과물의 신뢰도를 올리고, 모션과 GIF로 움직임을 더하고, 그림자와 그라데이션으로 입체감과 색감을 잡아주는 식입니다.

모든 플러그인을 한 번에 다 쓸 필요는 없습니다. 작업할 때마다 필요한 것만 골라 쓰는 것이 더 효율적이고, 한두 개씩 천천히 익혀가면서 본인 스타일에 맞는 도구를 찾아가면 됩니다.

피그마는 기본 기능만으로도 강력하지만, 플러그인을 더하면 그만큼 결과물의 완성도가 올라갑니다. 1편의 이미지·아이콘 플러그인과 함께 사용하면 작업 흐름이 훨씬 매끄러워질 것입니다.
