## 로고 디자이너에게 목업이 중요한 이유

로고를 그냥 흰 배경 위에 띄워서 클라이언트에게 보내면 "그냥 그림 파일"처럼 보입니다. 같은 로고라도 명함, 간판, 패키지 위에 얹어서 보여주면 "실제로 쓸 수 있는 브랜드"로 인식되죠. 시안 수락률이 확연히 달라집니다.

문제는 퀄리티 좋은 목업이 대부분 유료라는 것. 그래서 **상업적으로 사용 가능한 무료 목업 사이트 7곳**을 정리했습니다. 라이선스까지 확인해서 클라이언트 프로젝트에 그대로 써도 안전한 사이트들만 골랐습니다.

## 무료 목업 사이트 추천 7곳

### 1. Mockups Design

- URL: https://mockups-design.com/
- 강점: 패키징(병·박스·파우치·캔), 플라이어, 명함, 브로셔 등 카테고리 압도적
- 라이선스: 상업적 사용 OK, 출처 표기 불필요
- 주의: PSD 원본 파일 재배포·판매 금지

광고와 회원가입 없이 바로 다운로드 가능. 특히 화장품·식품 패키징 시안 만들 때 1순위로 추천합니다.

### 2. 프루티드 디자인 (Fruited Design)

- URL: https://fruited-design.kr/
- 강점: 한국어 UI, 한국 디자이너 감성에 맞는 자료
- 라이선스: 유료 자료 구매 시 상업·비상업 사용 라이선스 포함. 무료 자료는 다운로드 페이지의 사용 조건 확인 필요
- 주의: 재판매·재배포·재라이선스 금지

영어 사이트가 부담스러운 분들에게 추천. 한국 사업자에게 시안 보낼 때 톤이 잘 맞는 자료가 많습니다.

### 3. Mr.Mockup

- URL: https://mrmockup.com/free-mockups/
- 강점: 간판·사이니지, 의류, 문구류 등 다양한 카테고리, 매주 업데이트
- 라이선스(Freebies): 개인·상업 사용 OK, 출처 표기 불필요
- 주의: 원본 PSD 재배포·재판매 금지

간판이나 의류 목업 찾을 때 가장 먼저 들어가는 사이트. 무료인데 디테일이 유료 수준입니다.

### 4. Minimal Mockups

- URL: https://www.minimalmockups.com/
- 강점: 디바이스(아이폰·맥북 등) 목업, 미니멀한 스타일, 로고 전용 섹션
- 라이선스: 개인·상업 사용 OK
- 주의: 재판매·재배포 금지

홈페이지나 앱 UI 시안 보여줄 때, 그리고 미니멀한 로고 프레젠테이션할 때 좋습니다.

### 5. Mockup World

- URL: https://mockupworld.co/
- 강점: 여러 디자이너의 무료 목업을 한 곳에서 검색 가능
- 라이선스: **자료마다 다름** — 각 다운로드 페이지의 라이선스 명시 항목 확인 필수
- 주의: 일부 자료는 출처 표기 필요, 일부는 자유. 외부 사이트로 연결되는 자료 많음

"내가 원하는 카테고리가 어디 있지?" 할 때 검색용으로 쓰기 좋습니다. 단, 다운로드 전에 라이선스 한 번 더 확인하세요.

### 6. Mediamodifier

- URL: https://mediamodifier.com/
- 강점: 브라우저에서 바로 편집·생성, 포토샵 없이 사용 가능
- 라이선스: **무료(Starter) 플랜은 테스트용**. 상업적 사용 라이선스는 유료 Professional 플랜($12.50/월~)부터 제공
- 주의: 클라이언트 시안용으로는 유료 플랜 권장

포토샵 없을 때 빠른 내부 미리보기용으로 활용. 클라이언트에게 최종 시안 보낼 거면 유료 결제 또는 다른 사이트 추천.

### 7. Freepik

- URL: https://www.freepik.com/
- 강점: 자료 양이 압도적, 거의 모든 카테고리 커버
- 라이선스: 비회원은 하루 3회 다운로드, 무료 회원도 일일 제한 있음(정책 변동). 무료 자료 사용 시 출처 표기 의무. Premium은 무제한·출처 표기 면제
- 주의: 무료로 쓸 때는 반드시 Freepik 출처 표기

자료가 워낙 많아서 안 쓸 수 없는 사이트. 다만 무료로 클라이언트 작업에 쓰려면 출처 표기 부담이 있으니, 가급적 위 6개 사이트로 해결하고 Freepik은 보조로 쓰는 걸 추천합니다.

## 포토샵 목업 사용법 (3분 가이드)

1. **PSD 다운로드** — 보통 50~300MB 정도, 압축 풀어서 PSD 파일 엽니다.
2. **스마트 오브젝트 찾기** — 레이어 패널에서 "Your Design Here", "Place Your Logo" 같은 이름의 레이어를 찾습니다.
3. **더블클릭으로 열기** — 썸네일을 더블클릭하면 새 창이 열립니다.
4. **로고 삽입** — PNG·JPG·AI 파일을 드래그해서 넣고 위치·크기 조정.
5. **저장 (Ctrl/Cmd + S)** — 저장하면 원본 목업에 자동 적용됩니다.
6. **내보내기** — File → Export → JPG/PNG로 내보내서 클라이언트에게 전송.

처음엔 어렵게 느껴지지만, 한 번 해보면 다음부턴 1분 안에 끝납니다.

## 로고 디자이너 워크플로우 팁

- **배경·비율 통일**: 클라이언트에게 시안 3~5장 보낼 때 배경과 비율을 통일하면 훨씬 프로페셔널해 보입니다.
- **흑백·컬러 둘 다 제공**: 같은 로고를 흑백 명함 + 컬러 패키지 조합으로 보여주면 활용도 어필 가능.
- **1목업당 1로고**: 한 목업에 로고 여러 개 넣지 마세요. 산만해집니다.
- **카테고리 믹스**: 명함 + 간판 + 패키지처럼 다른 카테고리를 섞어서 "이 브랜드가 실제 세상에 적용된 모습"을 보여주세요.
- **출처 표기 의무 자료는 따로 관리**: Freepik 무료 자료처럼 출처 표기가 필요한 건 클라이언트 시안에서 분리해서 쓰는 게 안전합니다.

## 무료 목업 사용 시 주의사항

1. **개인 vs 상업 사용 구분 확인** — 같은 무료라도 상업 사용은 안 되는 경우 있음.
2. **출처 표기 의무 여부** — Freepik 무료 등은 표기 필수.
3. **원본 PSD 재배포 금지** — 위 사이트 모두 공통. 편집 후 결과물만 공유 가능.
4. **클라이언트 프로젝트엔 라이선스 명확한 사이트 우선** — Mockups Design, Mr.Mockup, Minimal Mockups가 가장 안전.

## 마무리

목업 사이트는 많지만, **상업적 사용 + 출처 표기 불필요 + 퀄리티**, 이 세 가지를 동시에 만족하는 사이트는 의외로 적습니다. 위 7곳 중에서도 **Mockups Design, Mr.Mockup, Minimal Mockups** 이 세 곳이 라이선스적으로 가장 안전합니다.

이 세 곳만 즐겨찾기 해놔도 로고 디자이너 부업의 시안 작업이 훨씬 편해질 거예요.
