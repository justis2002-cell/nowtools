# 챗지피티 이미지 업데이트 후기 - 제미나이 나노바나나와 비교

## 그동안의 이미지 생성 흐름
디자인 부업을 하면서 이미지 생성 AI를 자주 사용합니다. 쇼츠 만들 때 들어갈 이미지, 포트폴리오 작업의 시안 이미지, 로고 시안까지 다양한 곳에 필요하기 때문입니다.

그동안은 **제미나이의 나노바나나(Nano Banana)** 로 이미지를 생성해왔습니다. 사실 저는 제미나이를 평소에 **"잼민이"** 라고 부릅니다. 하도 제 말을 안 들어서요. 그래도 나노바나나의 이미지 품질이 좋아서 어쩔 수 없이 잼민이를 참고 써왔던 거죠.

## 잼민이(제미나이)의 답답한 점
제미나이로 작업하면서 답답했던 점이 몇 가지 있습니다.

**제 말을 제대로 못 알아듣습니다.** 분명 구체적으로 설명했는데 자기 멋대로 해석해서 엉뚱한 결과물을 만들어내는 경우가 많았습니다. 이건 그나마 다시 시키면 되는데, 진짜 문제는 따로 있었습니다.

**한 번 잘못 생성하면 계속 똑같이 잘못 생성합니다.** "이 부분을 이렇게 바꿔줘"라고 아무리 말해도 처음 만든 이미지의 잘못된 부분을 계속 반복합니다. 수정 요청이 누적될수록 점점 산으로 가는 경우가 많아서, 결국 새 창을 열고 처음부터 다시 시작하는 일이 반복됐습니다.

이름값 한다고 잼민이 답게 군다 싶은 순간이 한두 번이 아니었습니다.

## 챗지피티 이미지 업데이트, 진짜 좋아졌습니다
이번에 챗지피티가 업데이트되면서 이미지 생성 품질이 확 올라갔습니다. 사실 예전에 챗지피티 유료 결제를 했다가 끊었었는데, 마침 무료 한 달 이벤트가 있어서 다시 써봤습니다. 결과는 충격적이었습니다.

**한 창에서 모든 작업이 끝납니다.** 예를 들어 웹사이트 포트폴리오 작업을 한다고 칠게요. 브랜드명, 타겟 고객, 색상 톤, 분위기를 알려주면 챗지피티가 그 정보를 바탕으로 로고 시안도 만들어주고, 메인 이미지도 만들어줍니다. 따로 정리해서 다른 도구에 넘길 필요가 없습니다.

**대화 맥락을 기억합니다.** 이게 진짜 큰 차이입니다. 앞에서 주고받은 대화를 기반으로 이미지를 만들어주기 때문에, 매번 처음부터 설명할 필요가 없습니다. "아까 그 분위기에서 색상만 좀 바꿔줘" 같은 자연스러운 요청이 통합니다.

**제 말을 잘 알아듣습니다.** 잼민이와 가장 큰 차이입니다. 같은 프롬프트를 줘도 의도를 더 정확히 파악해서 결과물을 만들어줍니다.

## 쇼츠 작업 흐름이 바뀌었습니다
저는 쇼츠도 만드는데, 이미지 작업 흐름이 완전히 바뀌었습니다.

예전에는 나노바나나에서 이미지 따로 만들고, 대본은 다른 곳에서 정리하고, 두 작업을 왔다 갔다 했습니다. 지금은 챗지피티 한 창에서 대본을 수정하다가 "이 장면에 어울리는 이미지 만들어줘"라고 하면 지금까지 대화 내용을 토대로 알아서 프롬프트를 만들어 이미지를 생성해줍니다.

작업 흐름이 끊기지 않으니 같은 시간에 훨씬 많은 일을 할 수 있게 됐습니다. 결국 무료 한 달 이벤트가 끝나기 전에 다시 유료 결제를 해야겠다고 결심했습니다.

## 그런데 잼민이는 못 끊습니다
이쯤 되면 "그럼 제미나이 끊고 챗지피티만 쓰면 되겠네"라고 생각하실 텐데, 그게 또 그렇게 간단하지 않습니다.

제가 안티그래비티라는 도구를 사용하는데, 이게 구글 생태계라 제미나이 유료 플랜과 연동됩니다. 제미나이 유료를 결제하면 안티그래비티의 사용 한도도 늘어나는 구조입니다. 그래서 이미지 생성은 챗지피티가 압승이지만, 제미나이 유료도 끊을 수가 없는 상황입니다.

결국 둘 다 결제하는 슬픈 결말이 됐습니다. 구글이 자기들 서비스를 서로 묶어두는 방식은 진짜 무섭습니다.
